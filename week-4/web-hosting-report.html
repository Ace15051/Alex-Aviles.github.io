    Web Hosting Report

    What is a web server? Why would you need one?

    A web server is a computer that runs websites. It's a computer program that distributes web pages as they are requisitioned. The basic objective of the webserver is to store, process and deliver web pages to the users. You would need one to run your website so it is available on the internet via intercommunication through hyperlinks.

    plain why getting a domain name is not enough to put your site up on the web.

    The domain name secures the name or web address of your intended website to create but it will not make it available on the web. The webserver processes and reads the code to make the website actually available on the web.

    What service does a website hosting company provide?

    Web hosting is a service that allows organizations and individuals to post a website or web page onto the Internet. A web host, or web hosting service provider, is a business that provides the technologies and services needed for the website or webpage to be viewed on the Internet.

    What are the most important characteristics to look for in selecting a good web hosting company?

    Some of the most important characteristics to look for is Reliability, Speed, Space, Substantial Bandwidth, Support, Standard Pricing/ Payment Plans.

    Research companies that provide website hosting plans. On average, what is the price range (price per month) for hosting a website?
