    Domain Name Report

    What is an internet domain name?

    An internet domain name is an identification string that defines a realm of administrative autonomy, authority, or control within the internet. Domain names are used to identify internet resources with a text-based label that is more memorable than the numerical addresses used in the internet protocol.

    Why are domain names so important on the web?

    Websites are very dependent on Domain names as they are used by internet users to reach their information.

    Why do domain names have to be unique?

    Domain names are used to develop a unique identity to the webpage and can be used to make them more easily accessible.

    What does it mean to "register)" a domain name?

    It means once you come up with a website, you need to go through a Domain Registrar, which is a service accredited by the ICANN (Internet Corporation for Assigned Names and Numbers) that allows you to purchase domain names and checks their availability.

    If you wanted a domain name that is already taken, how could you get it?

    In order to register a domain name that is already owned, you must purchase it from the current owner making some of these domains very valuable to some companies.
